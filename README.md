# ServiceNow CMDB Baseline Diagnostic

**Framework completo de avaliação da maturidade e qualidade da CMDB** — com veredito GO/NO-GO claro para evolução de capacidades ITOM.

![Version](https://img.shields.io/badge/version-4.x-blue)
![ServiceNow](https://img.shields.io/badge/ServiceNow-Quebec%2B-brightgreen)

### ✨ Funcionalidades

- **Diagnóstico determinístico** por 9 domínios críticos (Probes, Sensors, Write Control, IRE, Reconciliation, Health, CSDM, etc.)
- Console offline bonita e interativa (HTML único)
- Veredito GO / GO-RESTRITO / NO-GO
- Score ponderado + Maturidade (1-5)
- Plano de correção priorizado automático
- Export para Word, Excel e PDF
- Comparativo antes × depois

---

### 📋 Como usar (Rápido)

1. Execute o script `CMDBBaselineDiagnostic_v4.js` em **Scripts - Background** (escopo Global + SNutils)
2. Copie todo o output
3. Abra o `cmdb_assessment_console_v2.html` no navegador
4. Cole e clique em **Analisar**
